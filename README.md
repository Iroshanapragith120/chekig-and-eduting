## Installation & Usage

1. Download & extract the repository.  
2. Open a terminal inside the repository and type:
   ```bash
   pip install -r requirements.txt
   ```
3. Then run:
   ```bash
   python main.py
   ```
   Select your service option and enter your link when prompted.  
4. The program will passively send the orders — you can leave it open.
